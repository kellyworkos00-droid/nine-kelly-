# kellyos Dashboard

A modern Next.js dashboard inspired by Modernize Next.js Dark, featuring Material UI, sidebar navigation, top navbar, dashboard stat cards, and basic authentication pages.

## Features
- Sidebar navigation
- Top navbar
- Dashboard stat cards
- Login and register pages
- Material UI dark theme

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure
- `src/components/Sidebar.tsx` - Sidebar navigation
- `src/components/Navbar.tsx` - Top navbar
- `src/components/DashboardCards.tsx` - Dashboard stat cards
- `src/components/Login.tsx` - Login page
- `src/components/Register.tsx` - Register page
- `src/App.tsx` - Main layout

## Customization
You can modify the sidebar, navbar, and dashboard cards in their respective component files.
